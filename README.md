# Team Knowledge Base

团队内部知识库系统

## 功能特性

- 富文本编辑器和 Markdown 两种编写模式自由切换
- 编辑内容自动保存草稿
- 文档按空间→目录→子文档的树形层级组织，支持拖拽调整
- 历史版本管理，支持 diff 对比和版本回滚
- 全文检索（标题和正文）
- 三级文档权限（公开、团队可见、仅指定成员）
- 多人实时协作编辑（基于 CRDT 算法）
- 实时同步和在线编辑者列表

## 快速开始

```bash
npm run install:all
npm run dev
```

- 前端: http://localhost:5173
- 后端: http://localhost:3001
